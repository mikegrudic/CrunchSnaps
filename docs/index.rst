CrunchSnaps
===========

CrunchSnaps is a Python package for making publication-quality visualizations
of hydrodynamic simulations, with a focus on GIZMO_ snapshot data.

.. _GIZMO: http://www.tapir.caltech.edu/~phopkins/Site/GIZMO.html
